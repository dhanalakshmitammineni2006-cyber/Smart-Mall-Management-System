#Smart Mall Management System

1.Overview:
The Smart Mall Management System is a web-based application developed using **Flask** that helps manage products and basic mall store operations efficiently. The system allows users to add, update, view, and manage product information through a simple and user-friendly interface.
This project aims to demonstrate backend development using Flask and database integration for managing real-world business data.

2.Features:
- Add new products to the system
- View product details
- Update product information
- Delete products
- Organized product data management
- Simple and responsive user interface

3.Technologies Used:
- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja Templates

4.Project Structure:

5.Installation and Setup:
  i. Create a virtual environment(python -m venv env)
  ii. Activate the virtual environment(env\Scripts\activate)
  iii. Install dependencies(pip install -r requirements.txt)
  iv. Run the application(python app.py)
  v.  Open in browser(http://127.0.0.1:5000)
